# Texas-hold-em-CLI
# 

## Description

This is a simple Command Line Texas Hold'em Poker game built using Python. The game allows a human player to play against a computer with basic poker rules such as dealing cards, betting, folding, and determining a winner.

## How to Run

1. Make sure Python is installed.
2. Download the project files.
3. Run the program:

```bash
    python main.py
```
## Features

* Human vs Computer gameplay
* Card dealing system
* Community cards (Flop, Turn, River)
* Simple betting system
* Winner determination
* Replay option

## Technologies Used

* Python
* Object Oriented Programming (OOP)
